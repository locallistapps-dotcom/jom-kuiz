==================================================
GOAL
==================================================

Upgrade the Parent & Child account system into the final production-ready architecture.

This module becomes the official account management system for Jom Kuiz.

==================================================
ACCOUNT TYPES
==================================================

Support two account types.

1. Parent Account
2. Child Account

Parent purchases the subscription.

Children never purchase subscriptions.

==================================================
PARENT LOGIN
==================================================

Parent login remains:

- Email
- Password

Parent profile contains:

- Full Name
- Email
- Phone (optional)
- Avatar (placeholder)
- Subscription Status (placeholder)
- Created Date

==================================================
CHILD ACCOUNT
==================================================

One parent can create multiple child accounts.

Each child belongs to exactly one parent.

Parent manages every child.

==================================================
STUDENT ID
==================================================

Every child automatically receives:

Student ID

Rules:

- Exactly 8 digits
- Numeric only
- Auto generated
- Unique
- Indexed
- Never changes
- Cannot be edited

Example:

48392715

==================================================
CHILD LOGIN
==================================================

Child login DOES NOT use email.

Child login requires:

- Student ID
- Username
- Password

Student ID + Username must match the same account.

If either is incorrect:

Return invalid credentials.
