# Web platform folder (placeholder)

This directory represents where Flutter's generated web host project
(`index.html`, `manifest.json`, icons) will live.

This Replit workspace does not run the Flutter web toolchain, so the full
platform project is not generated here. When you open this project in a
proper Flutter environment, run:

```sh
flutter create --platforms=web .
```

from the `jom_kuiz/` root to generate the real web project files.
