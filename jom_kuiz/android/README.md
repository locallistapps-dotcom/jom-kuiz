# Android platform folder (placeholder)

This directory represents where Flutter's generated Android host project
(`android/app`, `build.gradle`, `AndroidManifest.xml`, etc.) will live.

This Replit workspace does not run the Flutter/Android toolchain, so the full
platform project is not generated here. When you open this project in a
proper Flutter environment, run:

```sh
flutter create --platforms=android .
```

from the `jom_kuiz/` root to generate the real Android project files, then
apply this project's `applicationId`, app name, and icons.
