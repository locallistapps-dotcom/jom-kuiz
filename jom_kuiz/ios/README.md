# iOS platform folder (placeholder)

This directory represents where Flutter's generated iOS host project
(`ios/Runner.xcworkspace`, `Info.plist`, etc.) will live.

This Replit workspace does not run the Flutter/Xcode toolchain, so the full
platform project is not generated here. When you open this project in a
proper Flutter environment (with Xcode available), run:

```sh
flutter create --platforms=ios .
```

from the `jom_kuiz/` root to generate the real iOS project files, then apply
this project's bundle identifier, app name, and icons.
